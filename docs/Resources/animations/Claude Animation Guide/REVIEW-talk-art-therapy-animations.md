## Style Guide Lessons Learned

### Pattern Categories Demonstrated

1. **Horizontal Accumulation** (One Thread)
   - Use for: process, journey, building over time
   - Characters: `─ ┄ ╌ ═`
   - Motion: Horizontal progress with gentle vertical wave

2. **Boundary Awareness** (Frame & Beyond)
   - Use for: context, visible/invisible, liminal space
   - Characters: `┌ ┐ └ ┘ │ ─` + movement chars
   - Motion: Edge-to-center flow with breathing boundary

3. **Multiple Paths → Clarity** (Choosing the Stitch)
   - Use for: choice, discernment, agency, finding one's way
   - Characters: `+ × ┼ ╋ ┄ ╌`
   - Motion: Simultaneous paths with gradual opacity shifts

4. **Vertical Emergence** (Shadow Stories)
   - Use for: complexity→clarity, shadow→light, historical depth
   - Characters: `═ ━ ─ ┄ ╌ · ∙` (density progression)
   - Motion: Upward drift with pulsing gradient boundary

### Key Design Principles Confirmed

✅ **Crosses = decision points, intersections** — Use sparingly for moments of choice
✅ **Dashed/solid alternation = authentic stitching** — Creates in/out rhythm essential to textile work
✅ **Gradual opacity transitions = organic processes** — Never sudden; honor temporal patience
✅ **Persistent traces = integration not erasure** — What fades doesn't disappear; becomes context
✅ **Hand-worked imperfection** — Subtle waves, variations prevent mechanical feel
✅ **Horizontal characters preferred** — Maintain textile vocabulary; avoid circular/cellular patterns
