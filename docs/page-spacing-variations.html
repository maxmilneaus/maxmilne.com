Page Spacing Variations

Four systematic approaches to unify top spacing across Home, About, Now, and Projects pages. Each variation maintains the factor-based spacing system while creating different visual hierarchies.

Variation 1

Unified Baseline

All pages start at identical vertical position. Simplest, most consistent approach following Rams' "as little design as possible."

About 3.75rem
Now 3.75rem

Strengths

  • Perfect visual alignment when navigating between pages
  • Minimal cognitive load — one spacing rule for everything
  • Purest expression of systematic design
  • Easy to maintain and extend

Considerations

  • No differentiation between page types
  • Projects filter may feel cramped without extra breathing room

Strengths

  • About/Now get generous contemplative space
  • Density matches function (lists compact, narratives spacious)
  • All values still systematic (var(--space-xl), 1.5× factor)

Considerations

  • Most complex — different spacing for different page types
  • Requires careful documentation for future additions

Implementation Note

All variations use factor-based values from your existing spacing system (--space-xl = 3rem, or 1.5× factor = 4.5rem). The choice depends on whether you prioritize absolute consistency (Variations 1-2), functional grouping (Variation 3), or information hierarchy (Variation 4). For the Viola/Rams aesthetic, Variation 1 or 2 likely feels most aligned with "as little design as possible."
