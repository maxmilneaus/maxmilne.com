Filter Tags Feel Curated

Dieter Rams • Principle 1 · Score 7/10

Micro-interaction Score Bump
Now:

Clicking a filter tag swaps content instantly. Helpful, but the change feels binary—no sense of curation or intent.

After:

List items fade out then back in with a 50 ms stagger. The UI whispers “hand-picked” without slowing anyone down.

transition: opacity 300ms ease; transition-delay: calc(var(--item-index) * 50ms);

Keeps the calm aesthetic intact while introducing just enough flourish to register as innovative.

Instant Keyboard Shortcuts

Dieter Rams • Principle 2 · Score 9/10

UX polish Power Users
Now:

Navigation is purely click-based. Frequent visitors need multiple steps to jump between pillars (About, Now, Projects, Writing).

After:

Numbers 1–4 snap to the four main sections, and / primes the site search (once added). The UI shows care for habitual readers.

document.addEventListener('keydown', (e) => { if (e.key === '1') window.location = '/about'; });

No visual redesign required—just a small script that unlocks a better “tool” sensation.

Breadcrumbs Speak One Language

Dieter Rams • Principle 3 · Score 8/10

Quick win Consistency
Now:

The Topics page shows Max Milne / Writing and breaks the “•” separator rhythm used everywhere else.

After:

Swap to Max Milne • Writing. Two keystrokes later the site reads as one system again.

Max Milne • Writing

Micro change, macro signal: the design language is deliberate.

Now Page Sections Breathe

Dieter Rams • Principle 4 · Score 7/10

Quick win Readability
Now:

Headings like “Starting Talk. Art. Therapy.:” use punctuation and light weight that blur into the body copy.

After:

Remove trailing colons, bump weight to 600. Section breaks snap into focus without any visual clutter.

h4 { font-weight: 600; } // Remove trailing ':' in copy

Improves comprehension speed—especially vital on a “now” page where skimming is the default.

Navigation Confirms Where You Are

Dieter Rams • Principle 4 · Score 7/10

Quick win Orientation
Now:

Every nav link sits at the same weight. Visitors need to check the page title to confirm their location.

After:

Liquid logic adds an active class to the current page and slightly brightens the link. Navigation doubles as orientation.

{% if page.url contains '/about' %}class="active"{% endif %}

Zero layout change; pure reassurance that costs one condition per link.

Backlink Empty States Disappear

Dieter Rams • Principle 8 · Score 6/10

Quick win Calm Tech
Now:

Pages display “There are no notes linking to this note.” It’s technically honest but leaves a negative impression.

After:

Hide the block entirely when no backlinks exist. The page ends quietly, keeping attention on what is there.

{% if backlinks.size > 0 %} ...render section... {% endif %}

Calm technology principle: don’t surface absence when silence does the job.

Respect Reduced Motion Preference

Calm Technology • Respect the User

Accessibility Delight
Now:

Footer dash animation is subtle, but still runs for visitors who have asked macOS/iOS to reduce motion.

After:

A single media query pauses all non-essential animation for those users. Thoughtful and invisible.

@media (prefers-reduced-motion: reduce) { * { animation: none !important; } }

Signals care and keeps CSSDA judges happy without changing the default experience.

External Links Whisper, Not Shout

Dieter Rams • Principle 5 · Score 9/10

Tone Refinement
Now:

Every outbound link shows the ↗ icon permanently. It adds visual noise on dense writing pages.

After:

Fade the icon in on hover only. The cue is there when needed but doesn’t steal attention.

.external-link::after { opacity: 0; } .external-link:hover::after { opacity: 1; }

Preserves honesty while keeping typography as the hero.
